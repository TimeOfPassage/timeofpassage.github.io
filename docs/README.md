> 开阳(Mizar)是大熊座的一颗恒星，位在北斗的斗柄尾端第二颗星，是北斗七星之一。

- 与地球的距离： 82.84 光年
- 星等： 2.04
- 星座： 大熊座
- 坐标： 赤经 13h 23m 56s | 赤纬 +54° 55′ 31″
