# ffmpeg主要工作流程

读取文件 -> 解封装 -> 解码 -> 重新编码 -> 重新封装 -> 输出文件

# ffmpeg命令


ffmpeg命令主要分为六大部分

* ffmpeg信息查询部分
* 公共参数操作部分
* 文件主要操作参数部分
* 视频操作参数部分
* 音频操作参数部分
* 字幕操作参数部分
